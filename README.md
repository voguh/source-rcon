# Soruce RCON Protocol client for Java

This is a simple project that implements a [Source RCON Protocol](https://developer.valvesoftware.com/wiki/Source_RCON_Protocol) client for java.

> [!IMPORTANT]
> The development was migrated to Codeberg.
> The latest version of the project can be found at [https://codeberg.org/voguh/source-rcon](https://codeberg.org/voguh/source-rcon).
> This repository will no longer receive updates, but it will remain available for reference and historical purposes.
